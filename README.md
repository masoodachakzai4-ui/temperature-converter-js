Temperature Converter (JavaScript)
Description

This is a simple JavaScript program that converts temperature from Kelvin to:

Celsius
Fahrenheit
Newton scale
How it works
Starts with a Kelvin value
Converts it into Celsius
Then converts Celsius into Fahrenheit
Also converts Celsius into Newton scale
Uses Math.floor() to round down values
Formula Used
Celsius = Kelvin - 273
Fahrenheit = Celsius × (9/5) + 32
Newton = Celsius × (33/100)
Code Output

For Kelvin = 293:

Celsius = 20°C
Fahrenheit = 68°F
Newton = 6 N
